## Requirements

This project uses the [Gurobi Optimizer](https://www.gurobi.com) as its underlying solver. Gurobi is a commercial software, and you must have a valid license to run the code.

## Installation and Licensing

1.  **Obtain a Gurobi License**:
    *   **Academic Users**: Gurobi provides [free, full-featured academic licenses](https://www.gurobi.comacademia/academic-program-and-licenses/) for students, faculty, and staff at accredited institutions for non-commercial use.
    *   **Commercial Users/Other**: You will need to [purchase a commercial license](https://www.gurobi.comsolutions/licensing/).

2.  **Install Gurobi Software**: Download and install the Gurobi Optimizer software on your local machine from the [Gurobi website](https://www.gurobi.comdownloads/gurobi-optimizer-releases/).

3.  **Activate Your License**: Follow the official Gurobi instructions to activate your license, which typically involves running the `grbgetkey` command with your license key.

4.  **Install Python Package (if applicable)**: Ensure the `gurobipy` (or the relevant package for your language) is installed and can be imported in your environment.

For detailed instructions on installation and license setup, please refer to the [official Gurobi documentation](https://support.gurobi.com).
